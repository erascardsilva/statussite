# Status Site Monitor

Monitor de status para verificar disponibilidade de sites.

## Sobre

Aplicação desktop desenvolvida com Wails (Go + React) para monitorar o status de sites configurados.

## Como usar

### Desenvolvimento

```bash
wails dev
```

### Build

```bash
wails build
```

### Instalação

Depois de fazer o build:

```bash
./install.sh
```

### Desinstalar

```bash
./uninstall.sh
```

## Tecnologias

- Backend: Go
- Frontend: React + TypeScript
- Framework: Wails v2
